# Thư mục 3D Models cho AppGym

- Đặt các file 3D định dạng .glb vào đây (ví dụ: bench-press.glb, squat.glb, ...).
- Để hiển thị 3D động cho bài tập, chỉ cần map tên bài tập với file .glb trong app.js.
- Đã có sẵn file demo bench-press.glb để test thử.

Nguồn model miễn phí:
- https://poly.pizza/
- https://sketchfab.com/search?features=downloadable
- https://www.cgtrader.com/free-3d-models

Chỉ cần copy file .glb vào đây, app sẽ tự động nhận khi chọn đúng tên bài tập.
